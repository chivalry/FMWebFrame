Recommended Resources

Here are a few resources that we recommend for those new to the FileMaker API for PHP, or to PHP in general.

Web Publishing with PHP and FileMaker 9
by Jonathan Stark
This was one of the first books to be written about the FileMaker API for PHP. While it was written for FileMaker 9, it still serves as a great introduction to the FileMaker API for and to the PHP itself.

Web Technologies Forum
on FileMaker TechNet
This is a great online resource to turn to when you have questions about the FileMaker API for PHP.

Introduction to FileMaker Custom Web Publishing (CWP)
by David Nahodyl of Blue Feather
A lesson on how to create a PHP page that pulls information from a hosted database and displays it as an HTML page.
