fmQuickFind

Purpose:

Used to perform a Quick Find (a find across multiple fields on a layout).

Parameters:

[1] $connection: The FileMaker connection object to be used.
[2] $quickfind_layout: The name of the layout on which the Quick Find is to be performed.
[3] $query: The string to search for.
[4] $result_layout: The name of the layout from which results are to be obtained.
[5] $sort: Optional sort instructions. (See below.)
[6] $start_row: The starting record to be returned. Default is 1.
[7] $max_rows: The maximum number of records to be returned.

Output:

FileMaker_Result object

Example:

$quickfind_contacts = fmQuickFind (
   $fmDemo,
   'Contacts - Form',
   $_POST['q'],
   'Contacts - Form',
   'Last_Name ASC, First_Name ASC'
);

Notes:

To specify sort instructions, provide the field name to be sorted and the sort type (ASC / DESC). If sorting by multiple fields, comma-separate them. For example: Last_Name DESC, First_Name ASC
