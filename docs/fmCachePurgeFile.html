fmCachePurgeFile

"fmCachePurgeFile"

Purpose:

Deletes a specified file in the cache.

Parameters:

[1] $cache_file_name: The name of the file to be deleted.
[2] $cache_path: An optional path to use when deleting the file. Defaults to CACHE_PATH as set in the FMWebFrame settings file.

Output:

Returns TRUE if the file was deleted. Otherwise, FALSE.
