Additional Functions

FMWebFrame includes the following functions that we have found to be helpful.

fmDump
Wrapper for PHP's "var_dump" function.
Example: fmDump ( $fm_result, FALSE, 'Find Results' )

fmDumpAll
Uses "fmDump" to display several variable scopes, including _SERVER, _SESSION, _COOKIE, _GET, and _POST.

fmError
Maps a given FileMaker error code to its corresponding error message.
Example: fmError ( 202 ) = Field access is denied

fmGetDocumentRoot
Returns the document root directory under which the current script is executing, as defined in the server's configuration file.
Example: fmGetDocumentRoot () = /home/156317/domains/fmwebframe.com/html

fmGetRemoteAddress
Returns the IP address from which the user is viewing the current page.
Example: fmGetRemoteAddress () = 198.8.80.15

fmGetRequestTime
Returns the time that the request was made.
Example: fmGetRequestTime () = 1434420648

fmGetRequestURI
Returns the URI which was used to access this page.
Example: fmGetRequestURI () = /application/extra-functions.php

fmGetQueryString
Returns the query string, if any, with which the page was accessed.
Example: fmGetQueryString () =

fmGetServerAddress
Returns the IP address of the server on which the current PHP script is executing.
Example: fmGetServerAddress () = 205.186.187.224

fmGetServerName
Returns the name of the server host under which the current script is executing. If the script is running on a virtual host, this will be the value defined for that virtual host.
Example: fmGetServerName () = fmwebframe.com

fmGetServerPort
Returns the port on the server machine being used by the web server for communication. For default setups, this will be '80'; using SSL, for instance, will change this to whatever your defined secure HTTP port is (typically 443).
Example: fmGetServerPort () = 80

fmGetURL
Uses PHP's "curl" functionality to replicate FileMaker's "Insert From URL" script step.
Example: fmGetURL ( 'example.com' )

fmGetUUID
Returns a unique identifier based on the current time.
Example: fmGetUUID() = fmwf-557f85a9099d75-36657733

fmIsValidEmail
Used to determine whether a given value is a properly-formed email address. Returns a boolean value.
Example: fmIsValidEmail ( 'someone@testcom' ) =

fmIsValidURL
Used to determine whether a given value is a properly-formed URL. Returns a boolean value.
Example: fmIsValidURL ( 'www.test.com' ) =

fmStripTags
Returns the given value with unwanted characters either stripped or encoded.
Example: fmStripTags ( 'This is <bold>bold text</bold>. Or not.' ) = This is bold text. Or not.
